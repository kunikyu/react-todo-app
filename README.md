# TodoApp

React、TypeScript、Tailwind CSS を使用し、ローカルストレージでデータを永続化した「Todoアプリ」です。

## 開発履歴

- 2024-10/24：プロジェクト開始
- 2024-11/26：編集機能、嘘機能の追加
- 2024-11/27：嘘の強化
- 2024-11/27：このアプリについてを追加
- 2024-11/27：スタイルを強化
- 2024-11/27：レスポンシブデザインを強化
